Tool 1 — Image Compressor

Image Compressor

Reduce file sizes by up to 80% — no quality loss, no server uploads

The Image Compressor is the flagship tool on Toolkit Pro and the most widely used. It allows you to reduce the file size of any JPEG, PNG, WebP, or GIF image by up to 80% without any visible quality loss. Everything is processed using the browser's native Canvas API — your image data never leaves your device.

You can control three things: the compression quality (from 10% to 100%), the resize scale (from 10% to 100% of original dimensions), and the output format. The format selector lets you convert between JPEG, PNG, WebP, and the original format. This means you can turn a 2MB PNG into a 180KB WebP in about three seconds, making it the fastest image optimiser available without any software installation.

Tool 2 — Color Palette Extractor

Color Palette Extractor

Extract dominant HEX codes from any image instantly

The Color Palette Extractor is essential for designers, brand creators, and Pinterest creators who need to maintain visual consistency across their content. Upload any image — a product photo, a landscape, a screenshot — and the tool instantly extracts up to 8 dominant HEX colour codes from the image.

Click any swatch to copy its HEX code to your clipboard. Use the "Copy All HEX Codes" button to copy the complete palette at once. These codes can be pasted directly into Canva, Figma, CSS stylesheets, or any design tool. The extraction algorithm works by sampling the image at 80×80 resolution and grouping similar colours using a quantisation method — the result is an accurate representation of the image's true colour story.

Tool 4 — QR Code Generator

QR Code Generator

Generate high-precision scannable QR codes for any URL

The Privacy-First Architecture

One of the most important — yet least visible — aspects of Toolkit Pro is its architecture. Every tool on the platform uses one of the following browser APIs to process data locally without any server involvement:

  • Canvas API — Used by the Image Compressor, Color Extractor, Nature Designer, Art Studio, Voice Studio card renderer, and Gallery AI art generator. All pixel processing happens in memory on the client device.
  • MediaRecorder API — Used by Voice Studio to capture microphone input and encode audio in WebM format locally.
  • Web Audio API — Used by Voice Studio to generate background music synthesised in the browser using oscillators and noise buffers — no audio files are downloaded.
  • FileReader API — Used to read uploaded image files in the browser without uploading them to any server.
  • QRCode.js Library — Generates QR code matrices entirely client-side using the qrcodejs open-source library.
Privacy Guarantee: No image, voice recording, artwork, URL, or personal data is ever sent to Toolkit Pro's servers or any third-party server when using any tool. The only network requests made are for loading the page assets (HTML, CSS, fonts, libraries) and displaying Google AdSense advertisements.
